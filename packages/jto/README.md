# @json-to-office/jto

CLI and dev server for json-to-office with visual playground.

## Install

```bash
npm install -g @json-to-office/jto
```

## Usage

```bash
# Start dev server with visual playground
jto docx dev --input ./template.json

# Generate a file
jto docx generate --input ./template.json --output ./output.docx
```
